Create Lightning-powered blackjack tables for LNbits.

Its functions include:

- Creating blackjack dealers with configurable bets, decks, payouts, and rake
- Sharing public game pages for each dealer
- Accepting Lightning payments before a hand starts
- Letting players hit or stand from the browser
- Settling wins and pushes automatically through the dealer wallet
- Showing server seed details for a provably-fair style game flow

A focused game extension for operators who want to run simple public blackjack tables with Lightning payments and automatic sats payouts.
